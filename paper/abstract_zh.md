# 项目摘要

## 标题

注意力权重能反映 Token 重要性吗？基于 DistilBERT 的情感分类小规模研究

## 作者

- Tianhao Gong
- CHENKAI WANG

## 关键词

Transformer 可解释性，注意力机制，Token 重要性，模型解释，情感分类，DistilBERT

## 摘要

注意力权重常被用作一种方便的解释信号，用来说明 Transformer 预测时哪些词最重要，但已有研究也不断质疑它是否真的反映了 token 的实际重要性。在本项目中，我们在一个可控且聚焦的设定下重新检验这一问题：使用 DistilBERT 完成 SST-2 情感分类，并在四个随机种子下重复训练与分析。

对于验证集中的每个样本，我们分别使用最后一层 `[CLS]` 注意力、`gradient × input` 和 `leave-one-out (LOO)` 删除法对 token 进行排序。随后，我们通过 Spearman 相关系数比较不同排序之间的一致性，并通过 token 删除实验观察删除高排名 token 后模型置信度下降和预测翻转的情况。

实验结果表明，注意力与替代性重要性度量之间的一致性较弱：注意力与 `gradient × input` 的平均 Spearman 相关系数约为 0.22，与 `LOO` 的平均相关系数约为 0.18。删除实验进一步显示，`LOO` 选出的 token 会带来最大的置信度下降和最高的预测翻转率；基于注意力的删除虽然明显强于随机删除，但整体上仍弱于 `LOO`。这些发现说明，注意力确实包含部分有用信号，但不足以单独作为稳定可靠的 token 重要性代理。本项目因此提供了一个紧凑且可复现的实证基准，用于检验 Transformer 文本分类中关于可解释性的常见论断。
